# ZVM
 A simple hybrid stack + register based virtual machine built with C# which replicates Intel x86 instruction and register set. 
The performance isn't optimised whatsoever.

## What can it do?
It can:
 - **Add**
 - **Subtract**
 - **Multiply**
 - **Divide**
 - **Push**
 - **Pop**
 - **Move**
 - **Return**
 - **Call**
 - **Compare**
 - **Jump**
 - **Exit**
